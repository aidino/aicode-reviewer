Login Page Debug

Các bước để kiểm tra lỗi:

  1. Mở trang http://localhost:5174/login trong browser
  2. Mở Developer Tools (F12)
  3. Kiểm tra tab Console xem có lỗi JavaScript không
  4. Kiểm tra tab Network xem có resource nào fail load không
  5. Kiểm tra tab Elements xem DOM có được render không

Nguyên nhân có thể:

  • JavaScript Error: Component crash khi render
  • CSS Loading Issues: Stylesheet không load được
  • Import/Export Errors: Module không tìm thấy
  • AuthContext Issues: Context provider chưa setup đúng
  • Router Issues: Routing configuration sai
